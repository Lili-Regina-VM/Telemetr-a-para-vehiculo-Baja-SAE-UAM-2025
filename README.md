# Telemetría-para-vehículo-Baja-SAE-UAM-2025
# Autor: Lili Regina Vargas Mercado
Proyecto: Diseño e implementación de un sistema de telemetría para el vehículo Baja SAE UAM 2025.
Institución: Universidad Autónoma Metropolitana, Unidad Azcapotzalco.
Desarrollado como Proyecto de Integración en Ingeniería Electrónica.

Sistema de telemetría desarrollado para el vehículo Baja SAE UAM basado en ESP32, Raspberry Pi 5 y comunicación LoRa de largo alcance. El sistema permite transmitir, visualizar y almacenar variables del vehículo en tiempo real mediante Node-RED.
